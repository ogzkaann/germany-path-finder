# Product Spec

## Product

Germany Path Finder helps users in Germany explore source-backed residence and career transition paths.

## Audience

- International students.
- Job seekers.
- Ausbildung candidates.
- Skilled workers.
- People considering Chancenkarte, study applicant paths, Master's study, employment, or EU Blue Card paths.

## MVP Screens

1. Landing / Hero
2. Profile Builder
3. Personal Document Upload
4. Official Knowledge Vault
5. RAG Search / Ask
6. Path Result Screen
7. Local Progress Checklist

## Non-Goals

- No legal advice.
- No eligibility guarantee.
- No backend.
- No login.
- No payments.
- No automated government filing.
- No normal-user upload flow for official law or immigration PDFs.

## Success Criteria

- User can build and save a local profile.
- User can upload a PDF and review extracted structured fields.
- App generates and loads a manifest of official source entries.
- User can index owner-preloaded official files from the manifest.
- User can ask a question and receive citation-aware answers.
- User can evaluate placeholder path rules with fit, blockers, next actions, missing documents, and uncertainty.
